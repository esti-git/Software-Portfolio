# VolunteerElite 🤝

VolunteerElite
 היא מערכת חכמה ויעילה לניהול וחיפוש מתנדבים, המקשרת בין אנשים או ארגונים הזקוקים לעזרה לבין מתנדבים שרוצים לתרום לקהילה. המערכת מאפשרת ניהול דינמי של בקשות עזרה, סינונים מתקדמים ושיוך משימות בזמן אמת.

---

## 🚀 תכונות מרכזיות (Features)

- **ניהול ורישום מתנדבים:** רישום מהיר של מתנדבים חדשים למערכת ושמירת הפרטים שלהם בצורה מאובטחת.
- **לוח בקשות עזרה בזמן אמת:** הצגת כל המקרים הפתוחים שזקוקים לסיוע, כולל רמת דחיפות/חומרה ומיקום גאוגרפי.
- **מערכת סינונים מתקדמת:** אפשרות לסנן ולמיין את בקשות העזרה לפי קטגוריות, מיקומים ורמות דחיפות כדי למצוא את המשימה המתאימה ביותר בקלות.
- **תצוגת פרטים מורחבת:** לחיצה על כל בקשה פותחת חלון/עמוד עם כל הפרטים הרלוונטיים על המקרה.
- **מנגנון התנדבות חכם בלחיצה אחת ("אני רוצה להתנדב"):**
  - אם המתנדב טרם רשום במערכת – המערכת תפנה אותו אוטומטית לעמוד הרישום.
  - אם המתנדב כבר רשום – המערכת תשנה את סטטוס הבקשה ל"בטיפול" ותשייך אותה אליו באופן מיידי.

---

## 🛠 טכנולוגיות (Tech Stack)

המערכת בנויה בארכיטקטורת Full-Stack מלאה ומודרנית:

- **Frontend (צד לקוח):** HTML5, CSS3, JavaScript (Vanilla JS) – ממשק משתמש רספונסיבי, נקי ומהיר.
- **Backend (צד שרת):** Node.js בשילוב Express – ניהול ה-API, ניתובים (Routes) ולוגיקת המערכת.
- **Database (מסד נתונים):** MongoDB בשילוב Mongoose – שמירת נתוני המתנדבים ובקשות העזרה בצורה נומינלית וגמישה.
